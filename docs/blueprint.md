# **App Name**: RecruitTrack

## Core Features:

- User Authentication and Roles: Secure user login/signup with role-based access control (Admin, HR, Manager, Candidate).
- Job Management: Create, edit, and manage job postings with details, requirements, and status control.
- Candidate Management: Upload resumes, track candidates through hiring stages, and update candidate statuses.
- Resume Upload System: Upload and store resumes in Firebase Storage or AWS S3, with resume links visible in candidate details.
- Dashboard: Display key hiring metrics and optional simple charts for Admin/HR.
- AI Resume Matcher: An AI tool that automatically scores how well a candidate's resume matches a job description.

## Style Guidelines:

- Primary color: Dark, muted blue (#34495E) for a professional and trustworthy feel.
- Background color: Light gray (#ECF0F1), providing a clean, neutral backdrop.
- Accent color: Soft orange (#E67E22) to highlight key actions and information.
- Body and headline font: 'Inter', a grotesque sans-serif providing a modern and neutral look.
- Use a set of consistent and professional icons, with a style aligned to corporate UI, for navigation and status indication.
- Sidebar navigation for key modules (Dashboard, Jobs, Candidates, Settings), and table-based lists for job and candidate data, adhering to the design of Zoho Recruit.
- Use subtle transitions and animations for user interactions to improve usability.
